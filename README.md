# orbii-human-mimicking-robot
Mini robot that mimics human movement using MediaPipe and ESP32-based control.

## Features

- Human motion tracking using MediaPipe
- Omnidirectional ball-drive mobility
- Dual articulated arms
- Facial expression display
- Obstacle avoidance
- Audio feedback

## Hardware

- ESP32 controller
- MG90S servos
- Ultrasonic sensors
- Display module
- Camera module

## Software

- MediaPipe
- OpenCV
- Arduino Framework
